## Sound_recognition of Ruixinwei

### 项目简介
该项目旨在搭建一个性能鲁棒的神经网络，功能为对一段语音片段进行分类，
共有人声、包含背景音乐的人声、包含背景噪声的人声以及不包含人声四个类别。

### 项目环境
* `python3.x`
* `pytorch`
* `keras`
* ...
